Here's the file. To put it on GitHub Pages:

1. Create a repository

  • Go to github.com → New repository
  • Name it something like fiscal-policy-group
  • Set it to Public, check "Add a README"

2. Upload the file

  • Click "Add file" → "Upload files"
  • Drag in fiscal-policy-about.html
  • Rename it index.html before uploading (or rename after) — GitHub Pages serves index.html as the homepage

3. Enable GitHub Pages

  • Go to your repo's Settings → Pages
  • Under "Source," select Deploy from a branch
  • Choose main branch, / (root) folder → click Save

4. Your URL will be: https://YOUR-USERNAME.github.io/fiscal-policy-group/

GitHub takes 1–2 minutes to publish. Then paste that URL into your Facebook group's About section as the "Website" link. Done!
